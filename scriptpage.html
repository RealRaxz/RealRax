📜 Script Copy
🚀 Script นี้ถูกออกแบบมาเพื่อช่วยให้คุณเล่นเกมได้สะดวกและรวดเร็วขึ้น

✨ รายละเอียด
• รองรับการใช้งานบน Executor ส่วนใหญ่
• ทำงานได้บน PC และ Mobile
• ใช้งานง่าย เพียงคัดลอกแล้วนำไปวาง

📖 วิธีใช้งาน
1️⃣ กดปุ่ม Copy Script ด้านล่าง
2️⃣ เปิดโปรแกรม Executor ของคุณ
3️⃣ วาง Script ลงในช่อง Script
4️⃣ กด Execute หรือ Run Script
5️⃣ กลับเข้าเกมแล้วใช้งานเมนู Script ได้ทันที 🎮
