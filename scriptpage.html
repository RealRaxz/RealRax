📜 Script Copy
📝 คำอธิบาย

🚀 Script นี้ถูกออกแบบมาเพื่อช่วยให้คุณใช้งานฟีเจอร์ต่าง ๆ ในเกมได้ง่ายขึ้นและสะดวกมากขึ้น
⚡ รองรับการใช้งานบน Executor ส่วนใหญ่ทั้ง PC และ Mobile
💡 ใช้งานง่าย เพียงคัดลอก Script แล้วนำไปวางใน Executor ก็สามารถเริ่มใช้งานได้ทันที

📖 วิธีใช้

1️⃣ กดปุ่ม Copy Script ด้านล่าง
2️⃣ เปิดโปรแกรม Executor ของคุณ
3️⃣ วาง Script ลงในช่อง Script ของ Executor
4️⃣ กด Execute หรือ Run Script
5️⃣ กลับเข้าเกมแล้วใช้งานเมนู Script ได้ทันที 🎮
